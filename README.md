# HR-Analytics-Dashboard
This project presents an interactive HR Analytics Dashboard built in Microsoft Excel to analyze workforce metrics and employee attrition trends. The dashboard helps visualize key HR insights such as employee distribution, attrition patterns, performance ratings, and hiring trends to support data-driven HR decisions.
